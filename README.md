# openclaw-skill-cli-builder

Python CLI 工具脚手架生成。支持 Click / Typer / argparse，自动生成完整项目结构。

## 功能

- 项目脚手架自动生成（目录结构 + 代码模板）
- Click / Typer / argparse 框架选择
- 子命令支持
- pyproject.toml 配置（可安装 CLI）
- 进度条、彩色输出、配置文件支持

## 安装

将 `SKILL.md` 复制到 OpenClaw skills 目录即可。

## 使用场景

- "写一个 CLI"、"build CLI"
- 命令行工具开发
- Python 脚本转 CLI
- 脚手架生成
